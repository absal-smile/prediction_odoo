Smile Partner Mail History

This module allows you to consult all partner's received messages (Emails, Chatter Notifications, etc.) through their form view.

Usage

To use this module, follow these steps:

  1. Go to a partner Form view and click on the smart button History.
  2. You will be redirected to the partner's received messages.

Bug Tracker

Bugs are tracked on GitHub Issues. In case of trouble, please check there if your issue has already been reported. If you spotted it first, help us smash it by providing detailed and welcomed feedback here.
