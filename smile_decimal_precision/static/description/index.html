Display Decimal Precision

This module allows distinguishing computation digits and display digits in decimal precision. It is also possible to manage a number of decimal places to display on devices.

Features:

  • The administrator edits the decimal precision by specifying the name of the digits parameter.
  • The administrator sets the number of digits to compute.
  • The administrator sets the number of digits to display.
  • By default, the display digits equal the digits calculated.
  • The administrator can make the display digits greater than the digits calculated.
  • The administrator can make the display digits less than the digits calculated.
